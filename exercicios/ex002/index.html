Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio de par de tags <p> e </p>.

Se precisar quebrar o texto em algum lugar específico
como esse, você deve usar a tag <br>.


Para utilizar alguns símbolos como ® © € entre outros, é necessário utilizar um código para que apareça o símbolo requisitado. Ele começa com "&" seguido de um 'nome;' resultando no símbolo desejado.

Estes símbolos são chamados de "HTML Entities"

Exemplo:

® → '&' + 'reg;'
© → '&' + 'copy;'
™ → '&' + 'trade'
€ → '&' + 'euro'

Também é possível adicionar emojis no seu site utilizando códigos. Para acessar a lista de códigos entre em 'emojipedia.org'.

Após copiar o código do emoji desejado, substitua o "U+" por "&#x"

Exemplo:

'U+' + '1F604' → '&#x' + '1F604' → 😄
'U+' + '1F610' → '&#x' + '1F610' → 😐
'U+' + '1F435' → '&#x' + '1F435' → 🐵
