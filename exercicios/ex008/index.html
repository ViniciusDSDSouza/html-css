Códigos para formatação de texto

Negrito

Para escrever palavras em negrito utilizamos o código <b> (Não semântica)
Ou podemos Utilizar o código <strong> (Semântica)

Itálico

Para escrevermos textos em itálico utilizamos <i> (Não semântica)


Ou podemos utilizar <em> (Semântica)

Riscado

Para escrevermos o texto assim utlilizamos <del>

Sublinhado

Para digitarmos o Texto sublinhado utilizamos <u> (Não semântica)
Ou podemos utilizar a tag <ins> (Semântica)

Marcação

Para digitarmos o destacado desse jeitoutilizamos <mark>

Sobrescrito

Para digitarmos X20 + 3 utilizamos <sup>

Subscrito

Para digitarmos H2O utilizamos <sub>

Grande

Para digitarmos palavras GRANDES utilizamos <big>

Pequeno

Para digitarmos palavras menores digitamos <small>
